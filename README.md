# StockFlow 📦

A web-based inventory management system built with Python and Flask. Designed to track stock entries and exits in real time, organized by supplier invoices, work orders, and exchanges.

Built as my first full-stack project while learning backend development.

---

## Features

- Real-time stock tracking across multiple product categories
- Stock entries logged by supplier invoice number
- Stock exits by invoice, work order, or exchange
- Low stock alerts and minimum stock control
- Full movement history
- Reports and statistics dashboard

## Tech Stack

- **Backend** — Python, Flask, SQLAlchemy
- **Database** — SQLite
- **Frontend** — HTML, CSS, JavaScript
- **Deployment** — Railway

## Getting Started

```bash
# Clone the repository
git clone https://github.com/fernandochalbaud-png/StcokFlow.git
cd StcokFlow

# Install dependencies
python -m pip install -r requirements.txt

# Run the app
python app.py
```

Then open `http://127.0.0.1:5000` in your browser.
